# File structure

- **Stack**: Next.js 14 (App Router), JavaScript, Tailwind CSS
- **Path alias**: `@/` → `src/` (see `jsconfig.json`)

## Root

- `src/app/` — Routes and layouts
- `src/components/` — Reusable UI (layout, ui, welcome, dashboard, onboarding, scan)
- `src/lib/` — Constants
- `docs/` — Docs (structure, design, checklists)
- `public/` — Static assets

Details: **`docs/STRUCTURE.md`**. Pages/components: **`docs/PAGES_AND_COMPONENTS.md`**. Design: **`docs/DESIGN.md`**.
